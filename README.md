# CustomerAPI

Projeto **básico de estudos** criado com o objetivo de **treinar novas tecnologias** e consolidar fundamentos de desenvolvimento **backend e frontend**.

A principal motivação foi aprender **Fastify**, que eu ainda não havia utilizado, além de testar **Mongoose como ORM** no lugar do Prisma, utilizando **MongoDB** como banco de dados.

O projeto também conta com um **frontend simples e funcional**, desenvolvido para consumir a API.

---

## Objetivo do projeto

- Treinar o uso do **Fastify** em APIs REST
- Utilizar **Mongoose** como ORM no lugar do Prisma
- Trabalhar com **MongoDB**
- Praticar integração **Backend + Frontend**
- Criar uma interface rápida, simples e responsiva

> Projeto focado em **aprendizado e prática**, não em produção.

---

## Tecnologias utilizadas

### Backend
- Node.js
- TypeScript
- **Fastify**
- **MongoDB**
- **Mongoose (ORM)**

### Frontend
- **React**
- **Vite**
- TypeScript
- **Tailwind CSS**
- Abordagem **Mobile First**

---

## Funcionalidades

- Cadastro de clientes
- Listagem de clientes
- Atualização de clientes
- Remoção de clientes
- Integração frontend consumindo a API

---

## Rotas da API (exemplo)

- `GET /customers` → lista clientes
- `POST /customers` → cria um novo cliente
- `PUT /customers/:id` → atualiza um cliente
- `DELETE /customers/:id` → remove um cliente
